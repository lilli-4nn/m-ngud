# Pizza Maker Game

A simple HTML + JavaScript game where you run a pizza restaurant.

## How to Play

1. Open `index.html` in your web browser.
2. You start with €50.
3. Add toppings to your pizza (costs money per topping) - you'll see the pizza update visually with emojis.
4. Bake the pizza for €5 to prepare it.
5. Sell the baked pizza to waiting customers and earn €15-20.
6. Use earned money to buy new toppings in the shop.
7. Upgrade your restaurant to increase level and unlock more features.

## Features

- Prepare pizzas by adding toppings and baking them.
- Sell pizzas to customers.
- Buy new toppings like pesto, basil, chicken, fish.
- Upgrade your restaurant.

## Troubleshooting

- Ensure JavaScript is enabled in your browser.
- If buttons don't work, refresh the page.
